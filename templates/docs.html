What are these formats?

Video Formats: Common digital video containers supporting different codecs and quality levels.

Audio Formats: Various sound file types for music, voice, and other audio content.

Image Formats: Standard image file types for photos, graphics, and web images.

Office Formats: Microsoft Office and OpenDocument formats for everyday business documents.

Document & Publishing: Specialized formats for technical writing, academic publishing, and web content creation.

Data & Code: Formats for data exchange, coding notebooks, and technical documentation.
